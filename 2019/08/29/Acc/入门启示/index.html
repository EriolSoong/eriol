入门启示

述

听了耗叔的《左耳听风》的程序员练级攻略启蒙课程，他所推荐的是从web方向启蒙，而作为一个名义上的科班出生的我，基本没有接触过web方向，本科就是从C，C++入门，对于web后端的系统架构，前端的框架技术，基本都是听说止步，甚至一无所知。正是这样，我觉得《左耳听风》系列应该是对我整个技术栈的提升应该会有很大的帮助，十分期待这条路走下去会是一个什么样的结果。

悟

从web入门提升兴趣，要从两个不得不说的简单而又火爆的语言开始：Python和JavaScript。分别针对web的后端开发，和前端开发。
Python作为后端语言的后起之秀，目前的火爆程度可以这么来描述：即使是行外人，甚至都略知一二，这描述一点都不为过。Python具有良好活跃的开发社区环境，简单的语法，大量的第三方库，实用的语法糖，是编程入门的不二选择，但是就我看来作为解释型语言，效率上瓶颈是突破不了的，不可能和编译型语言相比，我觉得Python对于我本人的价值除了作为web开发的入门语言，再者应该是办公过程中提高效率的工具。
JavaScript作为前端语言，我上大学的时候就是前端必不可少的语言，过了这几年，JavaScript已经不仅仅是前端语言的存在，在各个行业都有他的身影。

知

  • Python入门书籍：
    1. 《Python编程快速上手》：偏应用
    2. 《Python编程：从入门到实践》：偏web
  • JavaScript入门书籍：
    1. 《MDN JavaScript官方教程》：官方，全面
    2. 《JavaScript 全栈教程（廖雪峰）》：偏应用
  • 前端知识：MDN知识库

行

博客系统

需求

  • 用户登录和注册（不需密码找回）。
  • 用户发贴（不需要支持富文本，只需要支持纯文本）。
  • 用户评论（不需要支持富文本，只需要支持纯文本）。

注意

  1. 用户登录时的密码不应该保存为明文，应该用 MD5+Salt 来保存（关于这个是什么，希望你能自行 Google）.
  2. 用户登录后，对于用户自己的贴子可以有 “重新编辑” 或 “删除” 的功能，但是无权编辑或删除其它用户的贴子。
  3. 数据库的设计，你需要三张表：用户表、文章表和评论表，它们之间是怎么关联的，你需要学习一下。这里有个 PHP 的 blog 教你怎么建表，你可以 前往一读。

其他

  • 图片验证码。
  • 上传图片。
  • 阻止用户在发文章或评论时输入带 HTML 或 JavaScript 的内容。
  • 防范 SQL 注入。参看PHP 官方文档 或 微软官方文档，或者你自己 Google 一下。
